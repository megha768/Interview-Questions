Q.53-What is the difference between html and xhtml?

Ans.- HTML(Hypertext Markup Language) and XHTML(Extensible Hypertext Markup Language) both are markup languages used for creating and displaying webpages. The main difference between them is the syntax and structure.HTML is more lenient in its syntax,while xhtml is an extension of html and has more strict syntax and follows XML rules.

Case sensitive - html is not case sensitive. xhtml is case sensitive, elements and attributes must be in lowercase.

Tag Closer - in html empty tags don't need to be closed. In xhtml even empty tags must be closed using '/'.

Attribute Values - in html attribute values can be without quotes in some cases.In xhtml attribute values must be enclosed in quotes.

Error Handling - In html browsers can often ignore errors. In xhtml errors are more likely to cause issues.

Extension - html file can have extension like - .html, .htm. XHTML file can have extension like - .xhtml, .xht, .xml
